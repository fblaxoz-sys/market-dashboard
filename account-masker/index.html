Account Masker
Runs on your computer only

Swaps real account numbers for fake tokens before a spreadsheet goes into an outside system, and swaps them back when the file comes out. Formulas, formatting, and everything else in the workbook are left exactly as they were.

Set up on Windows

  1. Unzip the folder somewhere permanent, like Documents or Desktop.
  2. Double-click Account Masker (Windows).bat.
    If a blue “Windows protected your PC” screen appears, click More info, then Run anyway — it asks once.
  3. First run only: press Y when it offers to install the free uv helper. The app window opens.

Set up on Mac

  1. Unzip the folder somewhere permanent, like Documents or Desktop.
  2. Install the free helper once — paste this in Terminal: curl -LsSf https://astral.sh/uv/install.sh | sh
  3. Right-click Account Masker.app → Open → Open.
    macOS asks once; after that, double-click works. On macOS 15 and newer, if there’s no Open button: double-click the app once (it gets blocked), then go to System Settings → Privacy & Security, scroll down, and click Open Anyway.

Protect the vault

After your first mask, a file called account_vault.json appears next to the app. It is the decoder that turns tokens back into real account numbers.

Never upload, email, or share it — and back it up somewhere safe. If it’s lost, masked files can’t be reverted. Each computer keeps its own vault, so files masked on one computer are unmasked on that same computer.

Nothing is sent anywhere — masking, unmasking, and the vault all stay on the computer where you run it. Full details in the README inside the download.
